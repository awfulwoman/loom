    mq

    @mixin mq($from, $until, $and, $media-type) { ... }

    Description

    Media Query mixin

    Lovingly repurposed from https://github.com/sass-mq/sass-mq

    Parameters

    parameterNameparameterDescriptionparameterTypeparameterDefault value
    $from

    (false) - One of $LOOM-mq-breakpoints

    		String or Boolean—none
    $until

    (false) - One of $LOOM-mq-breakpoints

    		String or Boolean—none
    $and

    (false) - Additional media query parameters

    		String or Boolean—none
    $media-type

    ($LOOM-mq-media-type) - Media type: screen, print

    		String—none

    Content

    This mixin allows extra content to be passed (through the @content directive).

    Example

    .element {
  @include mq($from: mobile) {
    color: red;
  }
  @include mq($until: tablet) {
    color: blue;
  }
  @include mq(mobile, tablet) {
    color: green;
  }
  @include mq($from: tablet, $and: '(orientation: landscape)') {
    color: teal;
  }
  @include mq(950px) {
    color: hotpink;
  }
  @include mq(tablet, $media-type: screen) {
    color: hotpink;
  }
}

    Requires

    • [variable] LOOM-mq-media-type
    • [variable] LOOM-mq-breakpoints
    • [function] convert-px-to-em
    • [function] get-mqbreakpoint

    Used by

    • [mixin] container
